# Imágenes de la Galería y Actividades

## Imágenes de la Galería

Para añadir las imágenes a la galería, coloca los siguientes archivos en este directorio:

1. **torrassa.jpg** - La Torrassa del Moro
2. **dolmen.jpg** - Dolmen de Llinars
3. **iglesia.jpg** - Església de Llinars
4. **castellnou.jpg** - Castellnou de Llinars
5. **auditori.jpg** - Auditori
6. **muntanyes.jpg** - Muntanyes del Vallès

## Imágenes de Actividades

Para las actividades, coloca los siguientes archivos:

1. **recollida-bolets.jpg** - Recollida de Bolets (cestas de setas)
2. **senderisme-montseny.jpg** - Senderisme al Montseny (excursionistas)
3. **torrassa-activitat.jpg** - Visita a la Torrassa del Moro (torre con ciclistas)
4. **castellnou-activitat.jpg** - Visita al Castellnou (edificio histórico con visitantes)
5. **gastronomia.jpg** - Gastronomia Local (mantener esta imagen)

## Formato recomendado:
- Formato: JPG o PNG
- Resolución: Mínimo 1200px de ancho para galería, 800px para actividades
- Tamaño: Optimizado para web (recomendado < 500KB por imagen)

Las imágenes se mostrarán automáticamente. Si alguna imagen no está disponible, se usará una imagen de respaldo de Unsplash.
